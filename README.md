## 🚀 Quick Start

### Option 1: View Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/zaynraiff-cmyk/metal-reapers-official-band.git
   cd metal-reapers-official-band
   ```
2. Open `index.html` in your browser
3. Add your `logo.png` to the root directory

### Option 2: Deploy to GitHub Pages
1. Go to repository **Settings** → **Pages**
2. Set **Source** to `main` branch
3. Your site will be live at: `https://zaynraiff-cmyk.github.io/metal-reapers-official-band`

## 📝 Customization

### Update Band Info
Edit `index.html` to customize:
- **Band name** (currently "Metal Reapers")
- **Member profiles** (Zayn, Ari)
- **Social links** (Spotify, TikTok)
- **Tagline & descriptions**

### Add Your Logo
1. Replace `logo.png` with your band's logo
2. Ensure it's a PNG file (280x280px recommended for hero section)

### Customize Colors
Edit the CSS color values in `index.html` `<style>` section:
- **Primary Red**: `#800c0c` and `#ff1a1a`
- **Background**: `#050505`
- **Text**: `#b0b0b0` and variants

### Update Social Links
Find and update these URLs:
```html
https://open.spotify.com/artist/YOUR_SPOTIFY_ID
https://www.tiktok.com/@YOUR_TIKTOK_HANDLE
```
